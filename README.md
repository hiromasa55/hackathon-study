# 🦊 ぺこナビ (PECO NAVI)

「学食で今日何食べる？」をAIが解決し、お財布と健康もサポートする学生向けメニュー提案アプリです。

## 🌟 プロジェクトの背景・概要
学食を利用する際、「何を食べたいか決まらない」「今月の予算に収まるか不安」「栄養が偏っていないか心配」と悩むことはありませんか？
「ぺこナビ」は、そんな学生のちょっとした気分や予算、前回までの食事履歴を考慮して、AIが最適な学食メニューを提案してくれるアプリです。

## 🎯 ターゲットと解決したい課題
* **ターゲット (WHO):** 学食を食べることは決めたが、気分・金銭・栄養面で何を食べようか決められない学生。
* **課題 (WHAT):**
  * 自分が何を食べたいのか、具体的に思い浮かばない。
  * 1ヶ月の使える予算の中で、賢く食費を管理するのが難しい。

## ✨ ぺこナビが提供する価値 (VALUE)
* **タイパ向上:** メニューを決める時間を大幅に短縮！
* **混雑緩和:** 迷う時間が減ることで、学食の食券機に並ぶ人の渋滞を改善。
* **健康サポート:** 栄養バランスが偏りがちな学生へのアプローチ。
* **お財布サポート:** 予算を考慮した金銭管理の実現。

## 🤖 AIの活用ポイント (AI ROLE)
学食の「通常メニュー」と「日替わりメニュー」の情報をAIに読み込ませています。
ユーザーのちょっとした「今の気分」を入力するだけで、AIが最適なメニューをチョイスします。また、昨日の食事内容や現在の予算額などを読み込ませることで、栄養面や金銭面も考慮した高度な提案を行います。

## 🚀 主な機能
* **気分でメニュー提案 (MVP機能):** 今の気分や食べたいものを入力すると、AIが学食メニューの中から最適なものを提案してくれます。
* **メニュー一覧 (Menu):** 学食で提供されているメニューを一覧で確認できます。
* **食事履歴＆集計 (History):** いつ、何を食べて、いくら使ったかを月ごとに確認可能。利用回数と合計金額を自動計算します。
* **設定 (Option):** プロフィール設定やページカラー設定など、アプリを自分好みに設定できます。
